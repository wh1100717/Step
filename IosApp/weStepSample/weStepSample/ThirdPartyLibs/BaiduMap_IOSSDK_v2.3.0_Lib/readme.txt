百度地图iOS SDK自v2.3.0起，采用可定制的形式为您提供开发包，当前开发包包含如下功能：
--------------------------------------------------------------------------------------
基础地图：包括基本矢量地图、卫星图、实时路况图和各种地图覆盖物，此外还包括各种与地图相关的操作和事件监听；
检索功能：包括POI检索，公交信息查询，路线规划，地理编码/反地理编码，在线建议查询，短串分享等；
LBS云检索：包括LBS云检索（周边、区域、城市内、详情）；
定位功能：获取当前位置信息；
计算工具：包括测距（两点之间距离）、坐标转换、调起百度地图导航等功能；
--------------------------------------------------------------------------------------
当前版本为v2.3.0，较上一个版本（v2.2.1）的更新内容如下：

新增
  可根据开发者的实际需求，下载满足需求的定制功能开发包
  
  地图和覆盖物
    包括基本矢量地图、卫星图、实时路况图、离线地图及各种地图覆盖物，此外还包括各种与地图相关的操作和事件监听
  检索功能
    包括POI检索、公交信息查询、路线规划、正向/反向地理编码、在线建议查询、短串分享等功能
  LBS云检索
    提供周边、区域、城市内、详情多种方式检索用户存储在LBS云内的自有数据
  定位
    提供便捷的接口，帮助用户快捷获取当前位置信息
  空间分析工具
    包括测距（两点之间地理距离）、坐标转换、调起百度地图导航等功能
